Key Valuation Metrics

We use standard discounted cash-flow techniques:

Net Present Value (NPV):
\[ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} \]

Economic Value Added (EVA):
\[ \text{EVA} = \text{NOPAT} - (\text{WACC} \times \text{Capital Employed}) \]

Internal Rate of Return (IRR) solves:
\[ 0 = \sum_{t=0}^{T} \frac{CF_t}{(1 + \text{IRR})^t} \]

Code Sample: NPV Sensitivity (Python)

Automating portfolio-level valuation scenarios


import numpy as np

cash_flows = np.array([120, 150, 180, 220])  # in ₹ Cr
discount_rates = [0.08, 0.10, 0.12]          # 8%, 10%, 12%

def npv(rate, cfs):
    """Simple NPV calculator."""
    periods = np.arange(1, len(cfs) + 1)
    return np.sum(cfs / (1 + rate) ** periods)

for r in discount_rates:
    value = npv(r, cash_flows)
    print(f"r = {r:.0%}  ->  NPV = ₹{value:,.2f} Cr")
