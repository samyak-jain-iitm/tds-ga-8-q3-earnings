Financial Formulas

**Net Present Value (NPV)** of future cash flows: \[ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} \]

**Return on Equity (ROE):** \[ \text{ROE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \]

**Cost-to-Income Ratio:** \[ \text{CTI} = \frac{\text{Operating Expenses}}{\text{Operating Income}} \]

Code: NPV Scenario Analysis (Python)

Example of automating earnings and valuation scenarios for the portfolio. 


import numpy as np

cash_flows = np.array([120, 150, 180, 220])  # in ₹ Cr
discount_rates = [0.08, 0.10, 0.12]          # 8%, 10%, 12%

def npv(rate, cfs):
    """Simple NPV calculator."""
    periods = np.arange(1, len(cfs) + 1)
    return np.sum(cfs / (1 + rate) ** periods)

for r in discount_rates:
    value = npv(r, cash_flows)
    print(f"r = {r:.0%} -> NPV = ₹{value:,.2f} Cr")
